# semanticHTML

Semantic Structure - tags:  \
	•	header: introductory information  \
	•	nav: nav links  \
	•	main: main content or body. only one main per page  \
	•	article: can stand independently of page or site. self-contained item for various contexts  \
	•	section: grouping nearby content with a theme. not self-contained but part of something  \
	•	aside: less important content. sidebars, or nonessential info, ads  \
	•	footer: footer. contact info, copyright, navs, etc.  \
  \
Semantic Text - tags: \
	•	H1,h2,h3,h4,h5,h6: headings. H1 is top-level. Usually only 1 h1 per page \
	•	p: standalone paragraph \
	•	a: hyperlink \
	•	ol: ordered list \
	•	ul: unordered list, no order \
	•	li: list \
	•	q: a short inline quotation of text \
	•	blockquote: long, multiline quotations \
	•	em: emphasis, text to be emphasized \
	•	Strong: strongly emphasized, or important text \
	•	code: block of computer code \
 \
Others:
	•	Abbr: abbreviation or acronym
	•	Address: contact info for author/owner of document
	•	caption: table caption
	•	cite: title of a creative work cited or quoted
	•	data: machine readable translation of given content
	•	Dialog: dialog box or window - element represents a modal or non-modal dialog box or other interactive component, such as a dismissible alert, inspector, or subwindow.
	•	figcaption: caption for a figure
	•	figure: self-contained content
	•	form: html input form
	•	legend: caption for fieldset tag
	•	fieldset: group related elements in a form
	•	mark: marked/highlighted text
	•	menu: unordered list
	•	noscript: alternate content for users that do not support client-side scripts
	•	optgroup: group of related option tags
	•	option: option in drop-down list (select tag)
	•	output: result of calculation
	•	progress: progress of a task
	•	search: defines a search section, for searching or filtering capabilities.
	•	small: smaller text
	•	sup: superscripted text
	•	sub: subscripted text
	•	summary: heading for a details element
	•	details: disclosure widget. Contents of summary are used as label.
	•	template: content to be hidden when page loads
	•	track: text caption tracks for video/audio
	•	hr: thematic break in content/topic. Often shown as horizontal line to break up page
